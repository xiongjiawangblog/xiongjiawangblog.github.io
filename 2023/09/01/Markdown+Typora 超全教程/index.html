Markdown+Typora 超全教程

😋 简要介绍

Markdown 是一种轻量型标记语言, 是一种语法. 以 .md 结尾的文本文件就是 Markdown 文件. 相较于 Word, 它更加像是 HTML 语言或是 LaTeX\LaTeXLATE​X , 并不是最淳朴的那种"所见即所得". 它处处透露着一种极简主义. 高效简洁清晰的同时, 又很简单. 看起来舒服, 语法简单, 尤其在处理纯文本上有很大的优势. 所有要写文本的时候都可以用上 Markdown! 它可以让你不再纠结什么字体, 什么样式, 什么排版. 而且逻辑清晰, 层次分明.

它相较于 Word, 兼容性非常高, 可以跨平台使用, 不用担心奇奇怪怪的版本兼容问题. 同时, 有许多网站都支持或正在使用 Markdown 语法. 如 Github (等一系列代码托管平台), StackOverflow(等答疑平台), 简书, 语雀 (等一系列笔记平台).

本介绍参考b站up主sakirary.此文档仅作为个人笔记, 无商业用途.

📐基础教程

标题有 * 表示该为扩展语法, 仅在 Typora 或 添加了扩展的 VS Code 本地生效, 在大多数平台上并不认可.

0. 前置准备

我们写文本的时候大多写的是中文, 可是输入法在输中文时使用的标点为全角标点, 如 ，。？！（）【】：；“”. 这些标点是不被 Markdown 所认可的, 也是无法转义的.

建议写 Markdown 的时候都用半角标点, 即英文标点, 如 ,.?!()[]:;"". 且每个半角标点在文本使用时加上后置空格, 符合英文标点的书写规范, 也更加美观.

以微软自带输入法举例, 在使用中文输入法时按下 Ctrl + .(这是个句号), 切换标点的全角与半角. 这样即可中文输入+半角标点.

1. 标题 [数个 “#” + 空格 前置]

(P.S. 标题会在目录与大纲分级显示, 可以跳转. 在 Typora 中建议开启 严格模式.)

2. 强调 [用 “**” 包围]

中国人民万岁!

3. 斜体 [用 “*” 包围]

中国人民万岁!

(P.S. 斜体并强调 [用 “***” 包围])

4. 删除线 [用 “~~” 包围]

法西斯

5. *高亮 [用 “==” 包围]

(注意: 此为扩展语法)

==中国人民万岁!==

6. 代码 [用 “`” 包围]

print(“Hello, world!\n")

7. 代码块 [三个 “`” +回车]

(P.S. 代码块里可以选择语言, 其会根据语言来自动高亮各个语句. 在选择语言后, ``` 会变为 ```` + 对应语言.)

8. 引用 [“>” + 空格 前置]

中国共产党万岁!

中国人民万岁!

(P.S. 引用是可以嵌套的.)

9. 无序列表 [“+” + 空格 前置]

  • 社会主义万岁!
  • 中国共产党万岁!
  • 中国人民万岁!

(P.S. 敲回车会自动补全, 敲回车后按下 Tab 键会缩进一级.)

10. 有序列表 [数字 + “.” + 空格 前置]

  1. 社会主义万岁!
  2. 中国共产党万岁!
  3. 中国人民万岁!

(P.S. 敲回车会自动补全, 敲回车后按下 Tab 会缩进一级. )

11. *上标 [用 “^” 包围]

(注意: 此为扩展语法)

2^10^=1024

12. *下标 [用 “~” 包围]

(注意: 此为扩展语法)

H~2~O是万物之源.

13. *注释 [“[^]” 后置]

(注意: 此为扩展语法)

人们宁愿去关心一个蹩脚电影演员的吃喝拉撒和鸡毛蒜皮，而不愿了解一个普通人波涛汹涌的内心世界.[1]

14. 链接 [用 “[ ]” + “( )” 分别包围文本与链接]

(注意: 文内跳转为扩展用法)

我的个人空间
[基础教程: 2. 强调](#2. 强调 [用 “**” 包围])

(P.S. 支持网页链接与文内跳转, 按住 Ctrl 并 单击鼠标左键 即可跳转.)

15. 任务列表 [“- [ ]” + 空格 前置]

  • [ ] 学习
  • [x] 起床

(P.S. 用 x 代替 [ ] 中的空格来勾选任务列表. 在 Typora 中可以直接用鼠标左键单击勾选框.)

16. 表格 [用 “|” 绘制表格边框]

学号 姓名 年龄

(P.S.填完一行数据后按ctrl+enter进入下一行)

17. 图片 [写地址或者复制粘贴]

我还是会选择拖进来或者复制粘贴啦~ 在 Typora 的设置里也可以改图片的储存方式.

18. 分割线 [三个 “-”+回车]

19. Emoji表情 [“:” 前置]

对于其余普通的 Markdown 文本编辑器, 可以直接将 Emoji 表情复制进来, 这是直接硬编码的 (刻进DNA里)

1. 内联 LaTeX\LaTeXLATE​X 公式 [用 “$” 包围]

(注意: 部分编译器会不识别部分符号)

LaTeX\LaTeXLATE​X 是最好用的论文排版语言! 不信你看!

an+bn=cna^n+b^n=c^nan+bn=cn

(P.S.用 $ 包围为单条公式, 按下两个 $ 并敲回车即生成公式块.)
